How to use it?

For now, there are just a few implementations

GET http://localhost:3000/api/recipes/

Returns a JSON object with all the recipes in the database

GET http://localhost:3000/api/recipes/vegetarian

Returns a JSON object with all the vegetarian recipes

GET http://localhost:3000/api/recipes/:id

Returns a JSON object with a specific recipe
